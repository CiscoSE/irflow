IR Flow - Incident Response Workflow for Threat Identification, Containment, and Research



Description

IR Flow allows a security analyst to automate and accelerate incident response procedures using the Cisco Security portfolio. The tool supports two distict modes, portal and headless. When running in portal mode, the analyst can query AMP and Cognitive Threat Analytics for current threats, use Investigate and Threat Grid to research them, and mitigate the threat using ISE, Firepower, and/or Umbrella. In headless mode, the tool can be configured to look for indications of compromize and quarantine affected hosts automatically.


Incident Response Workflow
1
Collect Indications of Compromise (AMP, CTA)
2
Quarantine Infected Hosts (ISE)
3
Find Malware Samples (Threat Grid)
4
Collect and Analyse Suspect Domains (Investigate)
5
Block Malicious Domains using Black List (Umbrella)
6
Add Custom Intelligence Feed (Firepower)
